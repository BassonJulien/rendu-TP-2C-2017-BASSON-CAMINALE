# rendu-TP-2C-2017-BASSON-CAMINALE

## Installation

To install the project, please follow the instructions :
--------------------------------------------------------

```
git clone git@github.com:chazzfit/rendu-TP-2C-2017-BASSON-CAMINALE.git
```
then :
-------

```
cd paths/rendu-TP-2C-2017-BASSON-CAMINALE
```

```
npm i
```

Start the project :
-------------------

```
npm start
```
